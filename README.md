# Operating-Systems

This repository holds my Operating Systems projects.

## How to Compile and Run				  

Use the shell to compile and run with commands in folder "Project <'project-num'>":
1. compile: <br />
	> make
2. run: <br />
	> ./P<'project-num'> config_<'file-num'>.config

OR: <br />
	> bash run.sh
